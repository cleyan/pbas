How to Run:

	1) Install Wampserver and put this repository in 'www' directory in C:drive.

	2) Run wampserver and confirm it by typing 'localhost' in the web browser's
           url box.

	3) To See webpage, type localhost/folder_name/login.php and press Enter. 
  	   In this case folder_name is "pbas".

	4) After pressing enter, a login page will be visible, check username and password 
	   in the PBAS database.

	5) Type Username and password and then hit "sign in" Button.

	6) Now explore "Research, publication And Academic Contribution" Area
	   for improvements.
